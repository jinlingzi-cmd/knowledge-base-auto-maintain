# 📋 模板说明

> 本文件夹存放所有笔记模板，帮助你快速创建标准化的笔记。  
> 使用方法：复制模板内容，粘贴到新笔记中，填写内容即可。

## 使用方式

### 方式一：手动复制（推荐，无需插件）
1. 打开模板文件
2. 复制全部内容
3. 创建新笔记，粘贴内容
4. 替换模板中的占位符（如 `[模型名称]`、`[公司名]`）

### 方式二：使用 Templater 插件（高级）
1. 安装 **Templater** 社区插件
2. 在模板中设置 Templater 文件夹为 `Templates/`
3. 使用 `Ctrl + P` → "Templater: Open Insert Template Modal"
4. 选择模板，自动插入并替换变量

### 方式三：使用核心模板插件（简单）
1. 在 Obsidian 设置中开启 **模板** 核心插件
2. 设置模板文件夹为 `Templates/`
3. 创建新笔记后，使用 `Ctrl + P` → "模板：插入模板"

## 模板变量说明

模板中包含以下占位符，使用时请替换：

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `[模型名称]` | 模型名称 | GPT-5、Claude 4 |
| `[公司名]` | 公司名称 | OpenAI、DeepSeek |
| `[领域]` | 所属领域 | 通用大模型、图像生成 |
| `YYYY-MM-DD` | 日期 | 2026-06-15 |
| `⭐` | 评分星级 | 根据实际评分填写 |

> 如果安装了 Templater 插件，`<% tp.date.now(...) %>` 等变量会自动替换为当前日期时间。
