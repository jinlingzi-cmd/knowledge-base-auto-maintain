# knowledge-base-auto-maintain
Automated monthly maintenance for Obsidian knowledge bases using Hermes Agent — parallel web search, dedup-aware patching, and push notifications.
A cron-driven workflow that keeps any Obsidian knowledge base up to date. Hermes Agent searches the web for recent releases, deduplicates against existing notes, patches files surgically, and delivers a monthly summary to WeChat or Telegram. Domain-agnostic — ships with templates for AI, tech, finance, and gaming.
